Software

SITH: If you are interested in cancer evolution, then check out my R/CRAN package SITH . This package implements a basic model of spatial growth and clonal heterogeneity. Interactive 3D visualizations possible with openGL. The user can also obtain synthetic DNA sequencing data sets (bulk or single-cell) from the simulated tumor, making it a potentially useful tool for validating methods.
